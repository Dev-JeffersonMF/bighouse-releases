# BigHouse — Releases

Repositório público de distribuição do instalador do sistema **BigHouse** para pizzarias.

## Como instalar

1. Vá em [Releases](https://github.com/Dev-JeffersonMF/bighouse-releases/releases/latest)
2. Baixe `BigHouseInstaller.exe`
3. Execute como **Administrador**
4. Siga as instruções na tela
5. O sistema inicia automaticamente na bandeja do Windows

## Como desinstalar

**Painel de Controle → Programas → BigHouse → Desinstalar**

## Atualizações automáticas (OTA)

O sistema verifica atualizações consultando `update.json` neste repositório.
Quando uma nova versão estiver disponível, o admin verá uma notificação no painel.
